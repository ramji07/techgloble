(1) introduction to Oracle Tools :

Oracle corporation is a company that produces the most widely used, server based Multi user RDBMS.

Oracle Engine :

The Oracle Engine compiles and executes oracle queries and commands. if the SQL queries and commands fails for any reason, oracle engine return an appropriate error message on prompt.

Oracle Tools :

The oracie product is primarily divided into -

  • Oracle Server Tools
  • Oracle Client Tools
    • (1) Oracle Server Tools :

    The oracle server is a program installed on the Server's Hard Disk Drive. This program must be loaded in RAM so that it can process user requests.

    This oracle server product is called Oracle Workgruup Server or Oracle Enterprise Server. The functionally of both these products is identical. However, the Oracle Workgroup Server restricts the number of concurrent users who can query the server. Oracle, Enterprise Server has no such restrictions.

    The Oracle Server takes care of the following-

    • Updating the database.

    • Retrieving information from the database.

    • Accepting query language statements.

    • Enforcing security specifications.

    • Enforcing data integrity specifications.

    • Enforcing transaction consistency.

    • Managing data sharing.

    • Optimizing queries.

    Managing system catalogs.

    Login identity and the password will allow user to connect to the oracle database. Once conn- ected to oracle database, a user can create or use the oracle resources like tables, views etc. depending on the permissions granted by DBA.

    (2) Oracle Client Tools :

    Once the oracle engine is loaded into server's memory, users would have to log into the engine to get work done. Oracle corporation has several client-based tools that facilitate this. The client tools most commonly used for Commercial Application Development is called Oracle Developer 2000.
