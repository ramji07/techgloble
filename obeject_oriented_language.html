(1) What is Object Oriented langauge

Until now, we have covered the basic concepts of object orientation at an abstract level. To use the concepts in practice in a database system,
however, we have to express them in some language. We can do this either of two ways :

1. We use the concepts of objects orientation purely as a design tool and encode them into, a relational database. We follow this
approach when we use entity relationship diagrams to model data and we then manually convert the diagram into set of relations.

2. We incorporate the concepts of object orientation into a language that we use to manipulate the database.
