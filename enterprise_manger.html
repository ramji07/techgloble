(1) Oracle Enterprise Manager :

It includes the server Manger Tool, which is used to startup the oracle Engine. Security Manager that allows DBA to create users and users permissions to use resources of the oracles database.

The Server Manager Tools :

The server manager tool is used to load the oracle engine from the server's hard disk drive into the server's memory. Oracle users will be allowed to access oracle resources only when the oracle engine is loaded in memory.

The Security Manager Tool :

Once the oracle engine is running, the security manager tool can be used by the DBA to create user accounts & grant permission to the user accounts. Creating of a user account will provide users with :

• Login Identity

• A Password and

• Permission to use oracle resources.
